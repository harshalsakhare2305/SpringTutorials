Spring HATEOAS — quick revision

Hypermedia As The Engine Of Application State

What it is

REST responses include _links telling the client what actions are possible next — like hyperlinks on a webpage.

"id":1,
"_links":{ "self":{"href":"/users/1"} }

Core classes

RepresentationModel<T>holds links
Linkrel + href
WebMvcLinkBuilderbuilds links
EntityModel<T>single item
CollectionModel<T>list
PagedModel<T>pagination

RepresentationModel in the model class

class UserModel extends RepresentationModel<UserModel> { Long id; String name; String email; } user.add(linkTo(methodOn(UserController.class) .getUser(id)).withSelfRel());

Self-referencing generic gives the model add() / getLinks() and auto _links serialization.

Generating Link with WebMvcLinkBuilder

1. linkTo(methodOn) recommended
linkTo(methodOn(UserController.class).getUser(id)).withSelfRel();
2. linkTo + slash
linkTo(UserController.class).slash(id).withSelfRel();
3. Direct Link.of()
Link.of("/users/"+id, "self");
